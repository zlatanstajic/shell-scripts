# Available scripts

Scripts you may use on any Unix-like system. Each links to its own page with
parameters, usage examples, and `.env` keys.

| Script | Description |
|--------|-------------|
| [Dev Setup]({{ site.baseurl }}/scripts/dev-setup/) | Development setup for git repositories |
| [PHP Switch]({{ site.baseurl }}/scripts/php-switch/) | Switch the main version of PHP on your OS |
| [Generate Password]({{ site.baseurl }}/scripts/generate-password/) | Generate a strong and secure password |
| [Backup]({{ site.baseurl }}/scripts/backup/) | Back up documents on a Linux machine |
| [Restore VS Code Folder]({{ site.baseurl }}/scripts/restore-vscode-folder/) | Restore a project's `.vscode` folder from backup |
| [Hash Filenames]({{ site.baseurl }}/scripts/hash-filenames/) | Rename files in a directory to random hashes |
| [Git Copy]({{ site.baseurl }}/scripts/git-copy/) | Copy the diff between two git commits, then zip it |
| [Splice Images]({{ site.baseurl }}/scripts/splice-images/) | Splice 2+ images horizontally via ffmpeg |
| [Splice Videos]({{ site.baseurl }}/scripts/splice-videos/) | Splice random clips of one video into a target-length output via ffmpeg |
